# PrivacyTap 威胁模型

## 保护资产

- Prompt 中五类业务敏感信息；
- Prompt 中误输入的认证凭证；
- 请求级占位符映射；
- 本地安全归档及可选观测输出。

## 信任边界

本机 PrivacyTap 进程与调用客户端属于信任域。第三方模型、外部观测系统、日志备份和普通运维访问者不应获得原始敏感值。

## 防御范围

- PII 在请求离开本机前替换；
- API Key 在请求上游前阻断；
- Vault 不序列化、不落盘、不上传；
- exporter 只接收脱敏安全事件；
- Authorization Header 可转发但不记录；
- 并发请求使用独立 Vault；
- exporter 异常不破坏模型调用。

## 非防御范围

- 本机被恶意软件控制；
- 管理员执行内存转储；
- 用户绕过代理；
- 图片、音频和附件中的隐私；
- 模型从公开信息推测真实身份；
- 上游自行生成新的敏感信息。

## 残余风险

- 规则可能漏报或误报；
- 匿名化可能影响必须读取原值的任务；
- 非 JSON 上游响应无法恢复；
- 首版不支持流式返回。

> PrivacyTap 用于降低敏感信息外发和进入观测系统的概率，不是法律合规认证产品。
